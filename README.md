# Digital Interactive College Magazine and Content Post System

## 📖 Overview

The Digital Interactive College Magazine and Content Post System is a web-based platform that transforms the traditional college magazine into a dynamic digital platform. It enables students, faculty, and administrators to publish, manage, and explore articles, campus news, achievements, blogs, and creative content in an interactive environment.

## 🎯 Objectives

* Provide a centralized platform for publishing college content.
* Encourage student participation and creativity.
* Enable online article submission and management.
* Replace traditional printed magazines with a digital solution.
* Improve accessibility and communication within the college community.

## ✨ Features

### Admin Module

* Secure Admin Login
* View Submitted Articles
* Accept/Reject Articles
* Publish URLs and Announcements
* Send Notifications
* Change Password

### Member Module

* User Registration with Email Verification
* Secure Login
* Submit Articles
* View Published Articles
* Receive Notifications
* Update Password

### Email Notification Module

* Sends notifications when articles are approved.
* Keeps users updated with newly published content.
* Improves user engagement.

## 🏗️ System Architecture

The system follows a three-tier architecture:

1. Presentation Layer (HTML, CSS, JavaScript)
2. Application Layer (JSP, Java)
3. Data Layer (MySQL Database)

## 🚀 Installation Steps

1. Install Java JDK.
2. Install Apache Tomcat Server.
3. Install MySQL Server.
4. Install NetBeans IDE.
5. Clone the repository:

   ```bash
   git clone https://github.com/your-username/digital-college-magazine.git
   ```
6. Import the project into NetBeans.
7. Configure MySQL database connection.
8. Start Apache Tomcat.
9. Run the project.

## 💾 Database

Create a MySQL database:

```sql
CREATE DATABASE newsportal;
```
